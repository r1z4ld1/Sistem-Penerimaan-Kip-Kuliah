#project pembuatan web responsive
Judul:Sistem Informasi Penerimaan Kip Kuliah
framework:laravel 12
backend : php
database : mysql
frontend : vite & tailwind css
package: spatie role&permission
 
#fitur-fitur utama
1. form login-Regiter
2. role & permission setiap fitur atau menu (contoh role mahasiswa bisa crud datanya sediri, tetapi diatur oleh admin di menu manajemen permission, 
role mahasiswa diberikan akses apa saja sesuai keinginan dari admin apakah hanya create dan read, atau read dan delete)
3. relasi database ERD
4. CRUD semua fitur dan ada permission (hak akses setiap role dan hanya admin yang atur hak ases apa yang dapat dilakukan setiap role) 

#fungsi utama web
1. fungsi css dan javascript (fungsi css & js harus berjalan saat web di hosting)
2. fungsi storage diweb contoh(gambar harus tampil pada saat web dihosting)
3. validasi form upload file dari sisi backend dan frontend
4. responsive desain di 3 kategori (smartphone, tablet/ipad laptop/pc)
5. struktur folder project harus rapi,sesuai dan mudah dibaca
